# 个人技术博客

这是一个基于Jekyll的个人技术博客，主要分享单片机开发和深度学习相关的内容。

## 特点

- 响应式设计
- 优雅的动画效果
- 清晰的文章分类
- 良好的阅读体验

## 本地运行

1. 安装Ruby和Jekyll
```bash
gem install bundler jekyll
```

2. 克隆仓库
```bash
git clone https://github.com/yourusername/yourusername.github.io.git
cd yourusername.github.io
```

3. 安装依赖
```bash
bundle install
```

4. 运行服务器
```bash
bundle exec jekyll serve
```

5. 访问 http://localhost:4000

## 部署到GitHub Pages

1. 创建一个新的仓库，命名为 `yourusername.github.io`
2. 推送代码到该仓库
3. 等待几分钟后访问 https://yourusername.github.io
